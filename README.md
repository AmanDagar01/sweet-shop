üöÄ Setup & Installation
Follow these steps to run the project locally.

Prerequisites
Python 3.9+

Node.js 18+

1. Clone the Repository
Bash

git clone <your-repo-url>
cd sweet-shop
2. Backend Setup
Navigate to the backend folder and set up the Python environment.

Bash

cd backend

# Create virtual environment
python -m venv venv

# Activate virtual environment
# Windows:
venv\Scripts\activate
# Mac/Linux:
source venv/bin/activate

# Install dependencies
pip install -r requirements.txt
# (If requirements.txt is missing, run: pip install fastapi uvicorn sqlalchemy passlib[bcrypt] python-jose[cryptography] multipart requests pytest httpx)

# Run the Server
uvicorn main:app --reload
The backend will run on http://localhost:8000

3. Frontend Setup
Open a new terminal, navigate to the frontend folder, and start the Next.js app.

Bash

cd frontend

# Install dependencies
npm install

# Run the development server
npm run dev
The frontend will run on http://localhost:3000

4. Seed Data (Optional)
To quickly populate your database with test users and sweets, run the seed script from the root directory:

Bash

python seed_data.py
